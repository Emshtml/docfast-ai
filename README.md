# 🚀 DocFast AI

**DocFast AI** é uma plataforma que gera **documentos profissionais automaticamente** usando inteligência artificial.

Contratos, notificações e declarações podem ser criados em segundos através de uma interface simples.

---

## 📌 Problema

Milhões de pessoas precisam criar documentos formais como:

* contratos
* notificações
* declarações
* cartas formais

Mas normalmente enfrentam problemas como:

* não saber escrever corretamente
* gastar dinheiro com advogados
* perder tempo procurando modelos na internet

---

## 💡 Solução

O **DocFast AI** resolve isso gerando documentos automaticamente.

O usuário apenas:

1. escolhe o tipo de documento
2. escreve os dados necessários
3. o sistema gera o documento pronto

Tudo em poucos segundos.

---

## 🚀 Deploy

Você pode colocar o projeto online em poucos minutos.

### 1️⃣ Clonar repositório

```
git clone https://github.com/seuusuario/docfast-ai
```

### 2️⃣ Entrar na pasta

```
cd docfast-ai
```

### 3️⃣ Deploy no Vercel

1. Acesse Vercel
2. Conecte o repositório GitHub
3. Deploy automático
